Overview

Tables can be used in a variety of ways to format your page. Some table layout tricks are discussed in this How_To. You can use more than one of them at a time for complex and interesting Web page layouts. Please note that some of the features discussed in this How_To cannot be viewed by all browsers; not all browsers support all features of tables. It's a good idea to test your documents with the browsers you expect your audience to be using.

Layout 1: Images Beside Text

A common layout trick is to construct a table that displays lines of text beside an image.
For example, if you want to display a list of your products and services beside your company logo, you can create a table that is two columns wide and has a row for each service that your company provides.
  1. Click on (the table button) in the tool palette.
  2. A box will be displayed prompting you to enter the number of rows and columns (Figure 1). Enter '5' in the rows field and '2' in the columns field, then click on Insert.

You can add rows and columns, delete rows and columns, expand cells, and contract cells using this dialog box. To create a single large row in the right-hand column, place your cursor in the top table cell in the second column of the table, and click the Extend Cell Down () button four times. At this point, you can type in text in the various table cells, and insert the logo image in the right-hand column. Text in tables may be modified as usual (bold, italics, header levels, etc.) The final product might look something like this in HoTMetaL PRO:

When you preview this in a Web browser, the tables borders, in general, will not be shown unless you turn them on. Therefore, all you'll see in most Web browsers is the text right beside the logo without any borders.
To have table borders on, place your cursor inside the table and choose Table Properties... from the Tools menu, and change the value for Border. The value is specified in pixels (zero means no borders). A table with a pixel border of one is shown in Figure 6.


Figure 6.

To create more spacing between the text and the image(s), you could add another column between the two columns using the Edit Table dialog box (place your cursor in the first column and click on the Insert Column Right button (). You can also specify a fixed pixel width for the first column. You can change the column width of any column by placing the cursor in any cell in that column and choosing Cell Properties... from the Tools menu. The following dialog box appears:


Figure 7.

Now, simply enter the spacing (in pixels) in the Width text box. You will not need to change the width for any other cells as long as they are in the same column. All cells in the same column will share the width of the widest cell.
You can change the vertical and horizontal alignment of text and images in table cells using the Vertical Alignment and Horizontal Alignment pull-down menus. For example, setting Vertical Alignment to Middle and Horizontal Alignment to Center would center the text or image in the table cell top-to-bottom and left-to-right.
Another trick! You can create a table that displays text on the left-hand side of the browser and the picture on the right-hand side, with blank space in between. This table will 'scale' to the size of the browser window; that is, the text and the picture will always be the maximum distance apart. Follow these instructions:
  1. Create a table with three columns and one row. Place an image in the left column, text in the right column and nothing in the middle column.
  2. Place your cursor inside the table and choose Table Properties... from the Tools menu. Set the Align attribute of the table to Justify.
  3. Next, place your cursor in any cell in the middle column and specify a value wider than the width of any screen (e.g., 1500 ) for the Width attribute. This blank cell will shrink to fit the width of your browser, placing the text of your document on the far right and the image on the far left.

Layout 2: Borders, Padding and Spacing

You can also add borders around tables, space inside table cells ('padding'), and spacing to add thickness to cell borders.
To change these attributes for the table:
  1. Place your cursor inside the table and choose Table Properties... from the Tools menu.
  2. Enter a numerical value (in pixels) for the Border, Cellpadding or Cellspacing text fields. Border has a maximum of 10. Figure 8 below shows the effects that can be achieved with these attributes in a table with one column and three rows.

Layout 3: BGColor

Background color is also a popular feature for page layout and can now be used for individual cells of tables. Currently this feature is supported only by Microsoft's Internet Explorer Web browser.
To add a background color for a particular cell:
  1. Place your cursor in a cell and choose Cell Properties... from the Tools menu. The dialog box for table cell attributes will be displayed.
  2. You can choose a background color using the color chooser. Clicking on the Choose... button will bring up the standard Windows color chooser. You can select a color from the predefined pallette, or create a color of your own. You could also enter the hexadecimal codes for a particular color in the BGColor text input. Figure 9 displays a three-row, one-column table with backgrounds of red (hex code is #FF0000), green (#00FF00), and blue (#0000FF).
You can also change colors of the entire table or of a row of the table. The selection of the colors is the same, but for a row or a table, set the BGColor attribute from the dialog box that comes up when you choose Row Properties... or Table Properties... from the Tools menu.

Layout 4: Indentation

You can also use tables to create indenting effects. (Other methods of creating indentation effects involve the use of definition, ordered, and unordered lists. However, these methods do create bullets, numbers and other formatting effects in most web browsers, and only offer limited spacing choices for indentation. In order to create more dramatic indentation authors sometimes 'nest' lists inside each other until the appropriate indentation is achieved. This can be confusing when you are coding, even in HoTMetaL PRO's interface, and the effects of nested lists can be somewhat unpredictable across different browsers.) To use tables for indentation, follow these instructions:
  1. Create a two-column table and put all of your information (text/graphics, etc.) in the second column, leaving the first column blank. This allows you to easily specify indentation without sacrificing readability or convenience.
  2. Specify indentation by placing your cursor in any table cell in the first column of your table and choosing Cell Properties... from the Tools menu.
  3. Type in the number of pixels you would like to indent under the Width attribute.
  4. Place your cursor inside the table, and choose Table Properties... from the Tools menu. Set the width of the table to 100%. This will make the table the same width as your Web browser document window.
